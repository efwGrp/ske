## ヘッダーのログアウトアイコン

一覧画面と入力画面のヘッダーにログアウトアイコンは設けている。
このアイコンによりログアウトを実行し、ログイン画面へ戻る。
定義画面によりアイコン非表示させることは可能。

以下のJquery式で特定可能。
```
#{defId}_header td:eq(2) i.bi-box-arrow-right
```
